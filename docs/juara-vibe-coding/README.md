# Juara Vibe Coding Optimization Docs

Folder ini berisi dokumen implementasi untuk mengoptimalkan Tax Feyments berdasarkan rubric #JuaraVibeCoding.

Gunakan dokumen ini sebagai pedoman kerja di Antigravity IDE atau AI coding agent lain.

## Urutan Baca

1. `01-problem.md` — memperkuat masalah, target audiens, dan urgency.
2. `02-solution.md` — memperkuat flow, UX, readiness score, dan output nyata.
3. `03-uniqueness.md` — memperkuat AI, OCR, what-if, dan wow factor.
4. `04-implementation-priority.md` — daftar pekerjaan prioritas P0/P1/P2.
5. `05-demo-script.md` — script video demo 2–3 menit.
6. `06-acceptance-criteria.md` — checklist final sebelum submit.
7. `07-ai-agent-instructions.md` — instruksi langsung untuk AI coding agent.

## Prinsip Utama

Tax Feyments harus terasa sebagai produk yang menyelesaikan masalah nyata, bukan sekadar kumpulan halaman dashboard.

Flow utama:

```text
Landing Page
→ Onboarding Persona
→ Input Data
→ Upload/Verify Document
→ SPT Readiness Score
→ AI Next Best Action
→ Tax Readiness Report PDF
```
