# 05 — Video Demo Script

Durasi target: 2–3 menit.

## 0:00–0:20 — Problem

Narasi:

```text
Banyak wajib pajak pribadi punya penghasilan dari gaji, freelance, atau UMKM, tetapi bingung dokumen apa yang harus disiapkan, apakah pajaknya sudah benar, dan apakah sudah siap lapor.
```

Tampilkan landing page dan pain point cards.

## 0:20–0:40 — Introduce Tax Feyments

Narasi:

```text
Tax Feyments adalah asisten persiapan pajak pribadi berbasis AI yang membantu pengguna merapikan data, mengecek kesiapan SPT, dan membuat ringkasan sebelum masuk ke sistem resmi DJP/Coretax.
```

Klik `Mulai Tax Checkup`.

## 0:40–1:05 — Persona Onboarding

Tampilkan pilihan persona.

Pilih:

```text
Karyawan + Freelancer
```

Narasi:

```text
Aplikasi menyesuaikan checklist berdasarkan persona pajak pengguna.
```

## 1:05–1:30 — Input dan Dokumen

Tampilkan data demo atau input cepat:

- gaji tahunan;
- penghasilan freelance;
- status PTKP;
- dokumen pendukung.

Jika OCR tersedia, tampilkan upload dan verifikasi manual.

## 1:30–2:00 — Readiness Score + AI Insight

Tampilkan dashboard:

```text
SPT Readiness: 72%
Tax Health: Perlu Perhatian
Dokumen kurang: bukti potong, daftar aset, rekap invoice freelance
```

Tampilkan AI insight:

```text
AI menyarankan pengguna melengkapi bukti potong dan mengonfirmasi apakah penghasilan freelance sudah dipotong pajak.
```

## 2:00–2:25 — What-if Simulation

Tampilkan simulasi:

```text
Jika penghasilan freelance naik 20%, estimasi pajak dan readiness berubah.
```

## 2:25–2:50 — Export Report

Klik export PDF.

Narasi:

```text
Output akhirnya adalah Tax Readiness Report yang berisi ringkasan data, estimasi pajak, dokumen yang kurang, dan langkah berikutnya.
```

## 2:50–3:00 — Closing

Narasi:

```text
Tax Feyments bukan pengganti DJP atau konsultan pajak, tetapi membantu wajib pajak menjadi lebih siap, rapi, dan percaya diri sebelum lapor.
```

Akhiri dengan hashtag:

```text
#JuaraVibeCoding
```
